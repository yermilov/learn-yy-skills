# home-it

**Home IT** skills for the `learn-yy-skills` marketplace — practical playbooks for the tech in your
home.

## Skills

- **check-network** — diagnose a slow home network end-to-end and recommend the highest-leverage
  fix. Measures from the affected device (throughput, idle-vs-loaded latency / bufferbloat, packet
  loss to the gateway, Wi-Fi signal and negotiated rate), localizes the bottleneck across four
  layers (ISP plan → router → Wi-Fi path → client), inspects the router admin UI, then ranks fixes —
  **wired backhaul first, a second wireless extender never**. Trigger on "make my home network/Wi-Fi
  faster", "speeds are below my plan", "slow at my desk", "should I buy an extender / new router?",
  "bufferbloat", «зроби домашню мережу швидшою», «інтернет повільний». Not for networks you don't
  administer, nor for server-side app latency.
